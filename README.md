## Hi, I'm Yusuf

I build practical software with a strong product sense: polished frontends, useful automations, AI-powered workflows, data tools, and dashboards that feel good to use. My work usually starts from a real-world problem, then becomes a deployable product with clean UI, fast iteration, and measurable value.
